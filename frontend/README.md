# Frontend для онлайн шахмат

React + TypeScript приложение для игры в шахматы.

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Откройте браузер на `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Структура проекта

- `src/api.ts` - API клиент для взаимодействия с бэкендом
- `src/components/` - React компоненты:
  - `Login.tsx` - Страница входа
  - `Register.tsx` - Страница регистрации
  - `Dashboard.tsx` - Главная страница с лентой игр
  - `Game.tsx` - Просмотр и игра в шахматы

## Особенности

- Аутентификация с JWT токенами
- Управление играми и приглашениями
- Просмотр истории игр и рейтинга
- Адаптивный дизайн
- Поддержка русского языка
